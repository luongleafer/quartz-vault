# Presentation Prep

- Topic: Ad blocker
- Arguments: Ad blocker help reduce distraction, annoyances and website's tracking
- Reason:
	- How Ad blocker works:
		- A list to filter ads
		- Block request to ads' server
		- Hide ads element
	- Ad blocker improves user experience
	- Ad blocker reduce traffic
- Problems:
	- Ad blocker can cause sites to disfunction
	- Reduce revenue to content creator
	- Big companies try to 'counter' the use of ad-blocker
- Solutions:
	- Allow user to customize filtered list and whitelist
	- Make ad easier to ignore/close
