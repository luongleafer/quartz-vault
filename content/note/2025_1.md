# Học kì 2025.1
## Tổng quan các học phần

```base
filters:
  and:
    - file.hasTag("university/subject")
    - semester == link("2025_1")
formulas:
  Tên: file.asLink(note["subject-name"])
  Untitled: ""
properties:
  note.credits:
    displayName: Tín chỉ
  note.subject-code:
    displayName: Mã học phần
  note.subject-name:
    displayName: Tên học phần
  formula.Tên:
    displayName: Tên học phần
views:
  - type: table
    name: Table
    order:
      - subject-code
      - formula.Tên
      - credits
    sort: []
    columnSize:
      formula.Tên: 447
      note.credits: 126
  - type: cards
    name: View
    order:
      - subject-name
      - subject-code
      - credits
    image: note.cover
    cardSize: 200
    imageFit: contain
    imageAspectRatio: 1

```
